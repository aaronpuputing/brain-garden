## 使用规则

1. **Agent 启动时**：先读此文件，如果"当前阶段"不是空，说明有未完成的门禁链
2. **开始新任务时**：清空旧记录，写入新的任务描述和门禁链
3. **每完成一个门禁**：更新对应行状态为 ✅，更新"当前阶段"为下一个门禁
4. **全部完成后**：在 CHANGELOG 中引用此文件的门禁记录
5. **如果 agent 中断或切换模型**：下一个 agent 读此文件即可接续，不依赖记忆
